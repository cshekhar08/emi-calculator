# File (Excel & PDF) Download Test Cases

---

## TC-FIL-001

**Title:** Verify Excel file downloads successfully  

**Steps:**  
1. Click “Download Excel” button.  

**Expected Result:**  
- Excel file downloads successfully.  
- File extension is .xlsx.  

---

## TC-FIL-002

**Title:** Validate Excel file data matches UI amortization table  

**Steps:**  
1. Download Excel file.  
2. Open file.  
3. Compare amortization values with UI.  

**Expected Result:**  
- Excel data matches UI amortization table exactly.
