# Chart & Table Validation Test Cases

---

## TC-CHART-001

**Title:** Validate Pie Chart values match principal and interest  

**Steps:**  
1. Capture Principal amount.  
2. Capture Total Interest.  
3. Observe Pie Chart segments.  

**Expected Result:**  
- Pie Chart proportions correspond to Principal and Interest values.  

---

## TC-CHART-002

**Title:** Validate Bar Chart matches amortization table  

**Steps:**  
1. Capture year-wise data from amortization table.  
2. Compare with Bar Chart values.  

**Expected Result:**  
- Bar Chart data matches amortization table values year-wise.  

---

## TC-CHART-003

**Title:** Verify chart (both pie and bar chart) updates after input change  

**Steps:**  
1. Modify Interest Rate.  
2. Observe chart refresh.  

**Expected Result:**  
- Both chart updates immediately.  
- No stale data remains.
